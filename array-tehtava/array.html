Kysymys : Mikä on taulukko (array) ja miten sellainen määritellään JavaScriptissa? Miten sitä käytetään?

Taulukkoa käytettään tallentamaan monta arvoa yhteen muuttujaan. Kutakin arvoa hyödynnetään
indeksin avulla.Taulukko luodaan esim
'let arrayNimi = {tähän laitetaan muuttujat pilkulla erotettuna}'

Array harjoitus katso js-tiedosto
1. Alkupeinen merkkijono - numerot pilkulla erotettuna
2. Merkkijonon numerot luettu taulukkoon ja tulostettu taulukon paikka index-3
3. Taulukko muutaman muutoksen jälkeen.
4. Lopullinen taulukko joka on järjestetty suuruusjärjestykseen
5. forEach() metodilla tulostettu taulukko


Osoitteessa https://www.w3schools.com/jsref/jsref_obj_array.asp on hyvä lista erillaisista taulukko toimininteista.

includes() - palauttaa true jos arvo löytyy taulukosta
indexOf() - paluuttaa indeksin arvon josta etsitty arvo löytyy. -1 jos ei löydy
reverse() - vaihtaa järjestyksen päinvastaiseksi
slice() - tekee uuden taulukon osajoukosta
paljon erillaisia metodeja jotka tutkivat taulukko kuten some()
toString() - muuttaa taulukon merkkijonoksi
